Wi-Fi Security Essentials

WEP (Wired Equivalent Privacy)

What it is: WEP was one of the earliest security protocols for wireless networks, aiming to provide security comparable to wired connections.

Vulnerability: With weak encryption keys and reused IVs, WEP keys could be quickly derived. Tools like Aircrack-ng made it easy for attackers.

Common Attacks: IV Replay, ARP Replay, FMS Attack

Tools: Aircrack-ng, Reaver, Fern Wi-Fi Cracker

How it got patched: WEP was replaced by WPA and eventually WPA2, as its fundamental flaws couldn’t be simply patched.

WPA (Wi-Fi Protected Access)

What it is: WPA was an interim solution replacing WEP. It introduced TKIP, making key changes more frequent.

Vulnerability: TKIP had weaknesses. Although more secure than WEP, WPA was still vulnerable to certain attacks.

Common Attacks: Handshake Capture, Beast Attack

Tools: Aircrack-ng, Cowpatty, Pyrit

How it got patched: WPA2 replaced WPA’s TKIP with AES, a stronger encryption method.

WPA2

What it is: WPA2 introduced AES encryption, making it much harder to crack than WPA or WEP.

Key security features: Strong AES encryption and improved authentication methods.

Common Attacks: KRACK, PMKID attacks

Tools: Aircrack-ng, hcxdumptool, hashcat

Notes: Pure WPA2 (AES-only) avoids vulnerabilities introduced by TKIP/WPA fallback modes.

WPA3

What it is: The latest standard, WPA3, enhances Wi-Fi security with robust encryption even if users choose weaker passwords.

Key improvements: Individualized data encryption, improved brute-force resistance.

Common Attacks: Dragonblood

Tools: hcxdumptool, hashcat

WPS (Wi-Fi Protected Setup)

What it is: Simplifies connecting devices without typing long passwords, using PINs or push-button methods.

Vulnerability: PIN-based WPS can be brute forced, making it a target.

Common Attacks: Pixie Dust Attack, WPS PIN brute force

Tools: Reaver, Pixiewps

Solution: Disable WPS if not needed.

MAC Filtering

What it is: Allows/denies devices by MAC address for added control.

Vulnerability: MAC addresses can be spoofed, so this is not a primary security measure.

Solution: Combine with WPA2 or WPA3 for stronger security.
